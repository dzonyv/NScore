# NScore

NScore is a web application for tracking live football scores and match updates.  
The project was built to provide a simple and clear way of displaying football data in real time through external API integration.

## Features
- Live football score updates
- Match information display
- Simple and responsive user interface
- Integration with external football data sources

## Technologies
- HTML
- CSS
- JavaScript
- REST API
- 
## Purpose
This project was created as a practical exercise in web development, API integration, and sports data presentation.
